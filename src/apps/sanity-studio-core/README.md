# `@monetis-io/santity-studio-core`

Sanity Studio to manage core data model.

## Prerequisites

- Required tools for this project:
  - Node.js
  - pnpm

## Environment variables

- `ENVIRONMENT` - runtime environment
  - `development`
  - `production`
- `PROJECT_ID` - Sanity project ID
- `DATASET` - Sanity dataset

## Local development

- Set the required [environment variables](#environment-variables)
- Run one of the following commands:
  - `pnpm start` to start the local development server
